**********************
Grobner-Shirshov-bases
**********************

A C++ library for calculating Grobner-Shirshov bases for involutive groups

===========
Environment
===========

* The project is developing under Eclipse CDT (Kepler)
* GCC 4.8.1
* GoogleTest 1.6.0

========
Building
========

::

  cd build
  ../build.h ..
  make
  ../bin/grobshir

=======
Testing
=======

::

  ../bin/GrobShir_test
